# Story-writer

This will be an AI story-writer, given a sentence he will compose a short story. Maybe he is not so accurate, but I am proud of me cause the sentence have a meaning😁😎😎🎉!!!


# Reasons
Made especially for other people to understand how AI and LLM’s work!!!
Basic lessons of theory and code on Colab. (the "lessons" are made about generative AI's, meaning they predict the next word creating sentences with punctuation.)
